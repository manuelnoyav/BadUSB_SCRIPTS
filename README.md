# BadUSB Scripts

Colección de scripts educativos para BadUSB (Ducky Script).  
Diseñados para entornos de aprendizaje y pruebas de seguridad autorizadas.

**No los utilices en sistemas sin permiso explícito.**

---

## Contenido

- `payload.txt` – Script para BadUSB. Abre PowerShell como administrador, descarga un script remoto y lo ejecuta.
- `main.ps1` – Script funcional.

## Responsabilidad

- **Uso autorizado:** Solo en equipos que poseas o con permiso expreso del propietario.
- **Webhooks y datos:** Si usas un webhook de Discord, créalo tú mismo y no lo compartas.
- **Archivos temporales:** Algunos scripts crean archivos temporales; revisa el código para entender su comportamiento.

---

## Personalización

Puedes modificar tanto el script PowerShell como el payload para adaptarlo a otras tareas de aprendizaje:
- Cambiar el destino del webhook o eliminar la parte de envío.
- Recolectar otra información (archivos, variables de entorno, etc.).
- Guardar resultados localmente en lugar de enviarlos.
